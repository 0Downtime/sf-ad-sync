# Code Of Conduct

## Our Standard

This project expects respectful, direct, and constructive collaboration.

Examples of expected behavior:
- focusing feedback on code, behavior, and evidence
- assuming good intent while still being technically rigorous
- helping keep discussions actionable and professional

Examples of unacceptable behavior:
- harassment or personal attacks
- discriminatory language
- doxxing or sharing private information
- disruptive behavior that prevents productive collaboration

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project spaces managed through GitHub.

## Enforcement

Project maintainers may remove comments, close discussions, or block participation for behavior that violates this code of conduct.

Report conduct concerns privately to the maintainer through GitHub.
